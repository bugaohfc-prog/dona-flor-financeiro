# FASE 9.5 — Empresa Switcher Foundation

Camada segura para preparar troca de empresa ativa no contexto SaaS.

## Incluído

- Listagem de empresas disponíveis para o usuário.
- Suporte a master user listar empresas.
- Persistência da empresa ativa já existente no AppContext.
- Troca segura de empresa recarregando dados do tenant.
- UI mínima apenas na área de usuários quando houver permissão e mais de uma empresa.

## Não incluído

- RLS obrigatório.
- Billing.
- Onboarding SaaS.
- Painel administrativo completo.
