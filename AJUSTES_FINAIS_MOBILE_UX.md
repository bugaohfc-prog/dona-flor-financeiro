# Ajustes finais mobile/UX

Alterações aplicadas nesta rodada:

- Dashboard mobile: cards de contas em aberto reorganizados com valor no topo direito e status/botão Pago alinhados lateralmente.
- Mobile alinhado ao padrão visual do desktop: cores mais suaves para pendente, vencido e pago.
- Página Contas mobile: cards suavizados com tons pastel e botões touch-friendly.
- Relatórios mobile: filtros de status em grid 2x2 para evitar 3 botões em cima e 1 embaixo.
- Lixeira: restauração agora pede confirmação antes de executar.
- Lixeira desktop/mobile: aviso de 60 dias reforçado visualmente.
- Usuários mobile: badges de perfil/status com sombra sutil para dar destaque ao status atual.

Build validado com `npm run build`.
